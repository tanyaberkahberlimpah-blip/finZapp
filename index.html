Integrasi Cloud Personal

Status Koneksi
❌ Tidak Terhubung

Catatan Penting:

  • Fitur ini mem-backup transaksi FinZ Anda ke Google Sheet milik Anda sendiri dalam format baris/kolom yang rapi.
  • Data Anda privat dan tidak digabung dengan database user lain.
  • Auto-Sync: Jika URL sudah tersimpan, aplikasi akan otomatis mem-backup data ke Cloud Anda tiap kali ada transaksi baru di latar belakang.
▶ Panduan Setup Cloud Personal
  1. Buka Google Spreadsheet baru.
  2. Klik Extensions > Apps Script.
  3. Klik tombol Salin Script di bawah & tempelkan.
  4. Klik Deploy > New Deployment.
  5. Pilih Type: Web App. Execute: Me. Access: Anyone.
  6. Salin Web App URL yang diberikan.
  7. Klik tombol Edit URL Cloud di atas dan tempelkan.
